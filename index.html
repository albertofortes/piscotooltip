A minimal customizable Tooltip in jQuery. Yep, another one.

Intro

Pisco Tooltip es un sencillo plugin de jQuery para crear tooltips. El funcionamiento es el mismo que todos: se muestra el contenido de la etiqueta title de una forma más atractiva y personalizada.

Pisco Tooltip usa el mínimo código js necesario para que sea muy fácil de modificar y ampliar si lo necesitas. Y usa un código html y css muy sencillo para que puedas usar tus propios diseños sin apenas conocimientos de CSS.

Simplemente un tooltip más, pero muy sencillo de usar.

Uso

El uso básico es muy simple, pon en la etiqueta title el texto que sea y este aparecerá en el Tootltip. Eso si, antes debes llamar a la función y a jQuery.

<a href="#" title="Este texto aparecerá en el tooltip" class="piscoTooltip" >link con tooltip</a>

Y llamamos al css, a la función y a los scripts, incluyendo jQuery:

<link rel="stylesheet" type="text/css" href="assets/css/pisco.tooltip.css" />
<script type="text/javascript" src="assets/js/general.js"></script>
<script type="text/javascript" src="assets/js/pisco.tooltip.js"></script>
<script>
$('.piscoTooltip').piscoToolTip();
</script>

Opciones y ejemplos de uso

Por defecto el tooltip se alinea debajo del elemento sobre el que se hace la burbuja, pero esto puede modificarse. Los valores por defecto son los siguientes:

id: 'piscoToolTip'

className: 'tooltip'

position: 'bottom'

separation: 10

delay: 0

Se pueden modificar fácilmente. Por ejemplo, un tootltip con la burbujita colocada a la derecha y con una id definida como 'bubble1':

link con tooltip con los valores por defecto modificados

La llamada al tooltip con las opciones modificadas sería:

<script>
$('.piscoTooltip-example1').piscoToolTip({
	position: 'right',
	id: 'bubble1',
	separation: 0,
	delay: 800
});
</script>

Diseños personalizados

Por supuesto puedes usar tus propios diseños. Tan sólo tienes que modificar el CSS.

link con tooltip a la derecha y diseño personalizado

Este ejemplo tiene simplemente el siguiente CCS:

#bubble2 {position: absolute;}
#bubble2 .cont {margin-left:15px; padding: 0 10px 0 0; line-height: 58px; background: url(../images/css/bg-bubble1.png) center right no-repeat; color:#8f9da4; font-size: 13px; border-radius:3px; font-weight: bold; text-shadow: 0 1px 1px #fff;}
#bubble2 .arrow {width: 15px; height: 58px; position: absolute; display: block; background: url(../images/css/arrow-bubble1.png) center left no-repeat; position: absolute; top: 0; left:0;}

Bastante sencillo, ¿verdad? La llamada a la función tampoco tiene misterio:

$('.piscoTooltip-example2').piscoToolTip({
	position: 'right',
	className: 'tooltip2',
	separation:0
});
